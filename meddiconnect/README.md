# MeddieConnect

MeddieConnect is a comprehensive healthcare platform that connects patients with doctors, facilitating seamless healthcare interactions and appointments.

## Features

- User and Doctor Authentication
- Doctor Search with multiple filters (specialty, area, insurance, ratings)
- Doctor Profiles with ratings and reviews
- Patient Medical Records Management
- Appointment Booking System
- Doctor Social Feed
- Interactive Maps for Clinic Locations
- Rating and Review System
- Medical Case Sharing Platform

## Tech Stack

- React with TypeScript
- Material-UI
- React Router
- Formik & Yup for form handling
- Google Maps API
- Styled Components
- Axios for API calls

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/         # Page components
├── services/      # API services
├── hooks/         # Custom React hooks
├── utils/         # Utility functions
├── types/         # TypeScript type definitions
├── assets/        # Static assets
└── context/       # React context providers
```
